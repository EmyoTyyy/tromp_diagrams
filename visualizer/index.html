Tromp Diagrams

multi-pane lambda calculus visualizer
scale: 8px
speed: 1.0x
Each pane is independent. Click + to add another. Use Sync in the toolbar to step / run all panes together — useful for comparing reduction strategies side-by-side. Multiline input is on by default (Ctrl+Enter to draw, Enter for newline). Long λ-bodies can be folded with the ⌄ button on each pane.
